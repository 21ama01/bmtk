class bmtk.simulator.utils.sim_validator.SimConfigValidator(schema, types=(), resolver=None, format_checker=None, file_formats=())[source]¶

Bases: jsonschema.validators.Draft4Validator

A JSON Schema validator class that will store a schema (passed into the constructor) and validate a json file.

It has all the functionality of the JSONSchema format, plus includes special types and parameters like making sure a value is a file or directory type, checking csv files, etc.

To Use: validator = SimConfigValidator(json_schema.json) validator.validate(file.json)
