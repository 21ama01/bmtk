bmtk.simulator.utils.tools.process_spikes.spikes_to_mean_f_rate(cells_f, spk_f, t_window, **kwargs)¶
cells_f: string
Full path to a file containing information about all cells (in particular, all cell IDs, and not just those that fired spikes in a simulation).
spk_f: string
Full path to a file containing cell IDs and spike times.
t_window: a tuple of two floats
Start and stop time for the window within which the firing rate is computed.

**kwargs spk_f_type: string with accepted values ‘txt’ or ‘h5’

Type of the file from which spike times should be extracted.

It is assumed here that TIME IS in ms and the RATES ARE RETURNED in Hz.

A dataframe containing a column of cell IDs and a column of corresponding average firing rates.

Usage: x = spikes_to_mean_f_rate(‘../network_model/cells.csv’, ‘output/spk.dat’, (500.0, 3000.0))
